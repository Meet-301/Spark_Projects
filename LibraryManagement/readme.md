# Library Management System (Console-Based)

A backend-focused Java application utilizing **JDBC** for persistent storage. This project demonstrates core **CRUD** operations and database connectivity without the abstraction of an ORM.

## ✨ Features
* **Add Book:** Register new titles into the database.
* **Remove Book:** Securely delete records from the inventory.
* **Issue Book:** Update book status and link to borrower records.
* **Return Book:** Reset book availability in the system.
* **Search Book:** Search a book based on book title.

## 🛠️ Tech Stack
* **Language:** Java
* **Database:** MySQL / PostgreSQL
* **Driver:** JDBC (Java Database Connectivity)
